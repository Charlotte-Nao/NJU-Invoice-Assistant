#  南大鲸麟战队智能报销助手 (云端工作站)

欢迎使用南大鲸麟战队专属的智能报销系统！
本系统已全面上云，无需安装任何软件或环境，只要有浏览器就能随时随地处理报销附件。

 **系统访问地址：** https://nju-invoice-assistant-2q2os016h-charlotte-naos-projects.vercel.app/

---

## 💡 使用指南

### 1. 进入专属仓库 (Workspace)
为了方便战队内部分组和个人管理，系统引入了**“仓库密钥”**机制。
* **默认仓库**：如果在上传时【仓库密钥】一栏留空，发票将默认存入公共的 `main` 仓库。
* **个人独立仓库**：在上传发票时，填入你自己设定的暗号（如姓名拼音 `zhangsan`）。发票会隔离进入你的专属仓库。在首页列表顶部输入该密钥点击“进入仓库”，即可查看你自己的数据。

### 2. 发票识别与上传
* 支持直接上传 **PDF 电子发票** 或 **JPG/PNG 截图**。
* 填写垫付人姓名、学号、工行卡号（系统会自动记住你的信息，下次免填）。
* 点击上传后，系统会自动提取发票号、计算含税金额及明细。

### 3. 补充截图附件 (重要！)
* 发票上传成功后，在下方的列表中展开该发票详情。
* 按照要求上传对应的 **“订单截图”** 和 **“支付截图”**。
* *注：系统支持一次性拖拽多张图片，并在前端自动进行极速压缩，速度极快且节省流量。*
* 上传完毕后，该条记录右上角会亮起绿色的 **“资料齐全”** 徽章。

### 4. 一键导出交付
* 当你的专属仓库里所有发票都显示“资料齐全”后，点击列表顶部的 **“导出 (ZIP)”**。
* 系统会自动生成一份严格符合南大财务要求的 13 列 Excel 汇总表。
* 所有的发票原件、订单/支付截图，都会被自动分装进以 `垫付人_日期_记录ID` 命名的独立文件夹中，并附带个人信息 TXT 文件。
* 直接将这个 ZIP 压缩包交给队长/财务对接人即可！

##  管理员/开发者指南 

本系统基于 Serverless 架构 (Vercel + Supabase) 构建。

**环境依赖说明：**
* 数据库迁移至 `Supabase PostgreSQL`。
* 文件存储迁移至 `Supabase Storage`。
* 发票解析引擎基于百度 AI 开放平台增值税发票原生接口，已移除本地 `Poppler` 等重型依赖。

**如何清空数据：**
在各个仓库列表顶部有红色的“一键清空记录”按钮。注意：此操作不可逆，将永久删除当前密钥仓库下的所有云端记录及图片，请在每次汇总导出完毕并核对无误后进行清理。
